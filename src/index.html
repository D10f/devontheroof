Particle Steering Behaviors

Steering behaviors algorithms are used to provide autonomous agents with realistic behavior that reacts to surrounding forces. They are simple to understand and effective in creating interesting movement patterns.

This demo was inspired by Daniel Shiffman to represent text that reacts to mouse movement.

Password Checker

Check your passwords against the popular "Have I Been Pwned?" website and find out if they've been leaked in any of the increasingly common data breaches. Seriously, start using better passwords!

You can provide passwords inline or through a CSV file, ideal if your password manager (such as KeePassXC) supports exporting data in CSV format. Your passwords will remain secured as only a computed hash is ever sent, as per the Have I Been Pwned API requires.
